# Contributing

Thanks for considering contributing!

## How to contribute
1. Fork the repo and create your branch from `main`.
2. Make your changes with clear commit messages.
3. Ensure the app starts with `npm start`.
4. Open a pull request describing your changes and why they are needed.

## Style
- Keep changes small and focused.
- Avoid adding large dependencies unless necessary.
- Prefer readable, maintainable code.

## Reporting issues
If you find a bug or want to request a feature, please open an issue with clear reproduction steps or a concrete proposal.
